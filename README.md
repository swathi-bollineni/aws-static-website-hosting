# 🌐 AWS Static Website Hosting using S3 & CloudFront

## 📌 Project Overview
This project demonstrates how to host a static website using **Amazon S3** and deliver it securely and efficiently using **Amazon CloudFront CDN**.

It showcases cloud deployment, content delivery optimization, and secure access configuration.

---

## 🚀 Features
- Static website hosting using Amazon S3
- Content delivery via CloudFront (CDN)
- Improved performance with caching
- Public access configuration using bucket policy
- Error handling and index routing

---

## 🛠️ Technologies Used
- AWS S3 (Storage & Hosting)
- AWS CloudFront (CDN)
- HTML, CSS (Frontend)
